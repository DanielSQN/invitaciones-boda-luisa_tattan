# Invitacion de boda Luisa & Tattan

Primera version de la invitacion web para Luisa y Tattan, construida con Next.js App Router.

## Desarrollo

```bash
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) para revisar la pagina.

## Scripts

```bash
npm run build
npm run lint
npm run start
```

## Version inicial

- Pantalla principal con monograma `L | T`.
- Carta/sobre interactivo con sello.
- Apertura animada al hacer clic.
- Fecha de boda: 26 de septiembre de 2026.
- Assets temporales en `public/` para textura de papel y flores decorativas.
